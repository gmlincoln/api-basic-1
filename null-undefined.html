The use cases of null and undefined

Undefined means the variable has been declared, but its value has not been initialize and null means an empty value. We can use null and undefined for different purposes and various situations.

Let's see some use cases of undefined and null.

Use Cases for undefined

Default Value
Set default value in a function parameter, this can be useful for setting default values in function parameters or when you want to check if a variable has been assigned a value yet.

Array Elements
When you access an array element that doesn't exist, it returns undefined. This can be used to check if an element exists in an array.
